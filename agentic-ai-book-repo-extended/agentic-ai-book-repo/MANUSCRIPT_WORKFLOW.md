# Manuscript Workflow

## Phase 1 — Scope
- finalize title and subtitle
- define audience
- fix part/chapter boundaries
- define what is in scope and out of scope

## Phase 2 — Structural drafting
- create chapter goals
- create reader outcomes
- create required tables and figures list
- identify glossary seed terms

## Phase 3 — Content drafting
- draft one chapter at a time
- keep sections structurally balanced
- record missing evidence and source needs immediately

## Phase 4 — Review
- run chapter review
- resolve terminology inconsistencies
- eliminate redundancy across chapters
- add missing trade-offs and failure modes

## Phase 5 — Publication readiness
- finalize references
- resolve placeholders
- verify legal and publisher data
- verify front matter and back matter completeness
- export build artifacts
