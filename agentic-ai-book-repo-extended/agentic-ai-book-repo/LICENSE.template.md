# License Decision Template

Choose and replace this file with the actual license for your repository and manuscript.

## Common patterns

### Code and scripts
- MIT
- Apache-2.0

### Book manuscript and documentation
- All rights reserved
- CC BY 4.0
- CC BY-NC 4.0

## Recommendation

Decide separately for:
- source code and scripts
- manuscript text
- figures and diagrams
- generated examples

Document the decision in:
- `LICENSE`
- `README.md`
- copyright page in the manuscript
