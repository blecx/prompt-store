# Directory Overview

## Top-level directories

- `manuscript/` — actual book files
- `templates/` — reusable Markdown templates
- `prompts/` — master, quick-use, and specialized prompts
- `review/` — checklists and review log
- `refs/` — source and reference management
- `docs/` — operating manuals
- `examples/` — copy-paste usage examples
- `scripts/` — helper utilities
- `dist/` — build output
- `.github/` — collaboration and automation
