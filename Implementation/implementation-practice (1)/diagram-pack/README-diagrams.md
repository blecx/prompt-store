# Diagram Pack

This pack contains the baseline diagram workflow for the local-first agentic AI system.

Included:
- editable Mermaid diagram sources
- prompt to generate or extend the diagram set

Baseline diagrams:
1. High-Level Agentic AI Architecture
2. Supervisor–Planner–Worker Orchestration
3. Planning Workflow
4. Tool Routing Architecture
5. Memory and Evaluation Architecture

Recommended extended diagrams:
- Agent runtime architecture
- MCP / A2A protocol layer
- Task-graph execution engine
- Memory internals
- Evaluation pipeline
- Governance / approval gates
- Local deployment topology
- Artifact generation flow
- Debugging / trace pipeline
- Regression testing workflow

Validation rules:
- verify every arrow
- verify no unintended connector merges
- verify title and short explanation
- keep Mermaid editable
- keep PNG readable
