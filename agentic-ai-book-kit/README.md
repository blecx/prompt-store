# Agentic AI Book Kit

This package contains a professional Markdown-based authoring kit for a technical book on Agentic AI.

## Files

- `agentic-ai-book-template.md`  
  Publication-oriented book skeleton including front matter, body structure, glossary, references, appendix, and readiness checklist.

- `agentic-ai-book-skill-prompt.md`  
  Reusable master prompt to steer book generation, revision, and critique toward technical-book quality.

- `agentic-ai-book-prompt-pack.md`  
  Specialized prompts for table of contents creation, chapter drafting, deepening sections, glossary work, references planning, and publication-readiness review.

- `agentic-ai-book-lector-prompt.md`  
  A strict reviewer / lector prompt for developmental and technical manuscript assessment.

## Recommended workflow

1. Start with the book template.
2. Use the book-skill prompt as persistent framing.
3. Generate or redesign the table of contents.
4. Draft one chapter at a time.
5. Deepen weak sections with the prompt pack.
6. Run the lector prompt after each major draft.
7. Maintain glossary, references, and appendices continuously.

## Suggested naming conventions

- `00-front-matter.md`
- `01-part-i-foundations.md`
- `02-part-ii-architecture.md`
- `03-part-iii-engineering.md`
- `04-part-iv-operations.md`
- `90-glossary.md`
- `91-references.md`
- `92-appendices.md`

## Practical note

For a real book project, split the large template into separate chapter files once the structure stabilizes.
