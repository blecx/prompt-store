# Prompt Selection Guide

## Use this prompt when...

### You need a full control layer

Use `prompts/core/report-skill-extended.md`

### You need a lighter control layer

Use `prompts/core/report-skill-compact.md`

### You need to convert existing analysis into a report

Use `prompts/quick-use/research-to-report-skill.md`

### You need to convert deep-research output into a report

Use `prompts/quick-use/deep-research-to-report-skill.md`

### You need a report outline

Use `prompts/specialized/outlines/report-outline-from-brief.md`

### You need a stronger executive summary

Use `prompts/specialized/executive-summary/executive-summary-builder.md`

### You need stronger findings or recommendations

Use the corresponding prompts in `prompts/specialized/sections/`

### You need review and criticism

Use the quality prompts in `prompts/specialized/quality/`
