# Directory Overview

## `docs/`
Guides and operational instructions.

## `templates/`
Reusable report and section templates.

## `prompts/`
Prompt library for generation, transformation, and review.

## `report/`
Split report skeleton to store the working manuscript.

## `review/`
Review prompts, checklist, and review log.

## `refs/`
Bibliography and evidence tracking.

## `scripts/`
Utility scripts for assembly and validation.
