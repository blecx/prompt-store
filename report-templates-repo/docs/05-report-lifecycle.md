# Report Lifecycle

## Lifecycle phases

1. assignment intake
2. research design
3. source gathering
4. synthesis
5. report drafting
6. review and correction
7. sign-off and distribution
8. versioned maintenance

## Why lifecycle matters

Many poor reports fail because drafting starts before the assignment and evidence model are clear. This repository tries to prevent that failure.

## Maintenance after release

Track:

- version number
- changed assumptions
- new evidence
- revised recommendations
- changed risk posture
