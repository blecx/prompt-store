# Quality Gates

A report should not be considered complete until it passes these gates.

## Gate 1 — Structural completeness

The report contains:

- purpose
- audience
- scope
- method
- findings
- analysis
- recommendations
- risks/limitations
- references

## Gate 2 — Evidence control

The report states:

- evidence basis
- confidence level where relevant
- assumptions
- unresolved gaps

## Gate 3 — Recommendation rigor

Every recommendation should have:

- rationale
- expected benefit
- trade-offs
- feasibility note
- owner or stakeholder implication

## Gate 4 — Consistency

The executive summary, findings, and recommendations do not contradict each other.

## Gate 5 — Audience fit

The report is understandable and useful to the target audience.
