# Report Outline from Brief

Create a detailed report outline from the provided assignment brief.

The outline must:

- fit the target audience
- support the intended decision or outcome
- include front matter, core analytical sections, and back matter
- avoid missing report-critical sections such as method, limitations, and recommendations
- include suggested tables and appendices where useful

Output:

1. proposed report title
2. report objective
3. audience statement
4. detailed table of contents
5. rationale for the structure
6. likely evidence-heavy sections
7. likely risk sections
