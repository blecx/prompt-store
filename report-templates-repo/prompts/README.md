# Prompt Catalog

## Core

- `core/report-skill-extended.md` — strongest general-purpose control prompt
- `core/report-skill-compact.md` — lighter version for shorter jobs

## Quick-use

- `quick-use/research-to-report-skill.md` — transform generic research into a report
- `quick-use/deep-research-to-report-skill.md` — transform deep-research output into a report
- `quick-use/report-post-formatter.md` — normalize and strengthen an existing draft

## Specialized

- `specialized/outlines/` — report outlines and redesign
- `specialized/deep-research/` — deep-research request and gap analysis
- `specialized/executive-summary/` — executive summary improvement
- `specialized/sections/` — build or repair specific report sections
- `specialized/report-types/` — prompts tailored to common report families
- `specialized/quality/` — review and defect-finding prompts
- `specialized/research/` — source-log and citation prompts
- `specialized/restructuring/` — gap analysis and board-version conversion
