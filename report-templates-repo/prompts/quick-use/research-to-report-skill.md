# Research-to-Report Skill

Take the provided research, analysis, or notes and convert it into a **professional report**.

## Requirements

- do not merely summarize
- impose report structure
- separate findings from interpretation
- make assumptions explicit
- expose evidence gaps and unresolved issues
- add recommendations only where justified by the material
- make the output useful for a real decision-maker

## Target structure

1. Executive Summary
2. Context, Objective, and Scope
3. Methodology / Input Basis
4. Key Findings
5. Analysis and Implications
6. Options and Trade-offs
7. Recommendations
8. Roadmap / Next Steps
9. Risks and Limitations
10. References / Placeholders
